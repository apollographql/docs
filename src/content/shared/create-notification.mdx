1. Go to your graph's **Settings** page in [GraphOS Studio](https://studio.apollographql.com/?referrer=docs-content).
2. Select the **Reporting** tab.
3. Click **Add notification** in the upper right.
4. Select a notification type and click **Next**.
5. In the dropdown, select which graph variant you want to receive notifications for.
6. Select an existing channel to send notifications to, or select which type of new channel you want to configure. Click **Next**.
7. If you're configuring a new channel, complete the steps in the next section.
