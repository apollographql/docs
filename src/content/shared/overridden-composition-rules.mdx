<RuleExpansionPanel
  title="OVERRIDE_DIRECTIVE_CAN_BE_REMOVED"
  whatItDoes="Checks that a field with the `@override` directive no longer exists in a source subgraph."
  rationale="If a field with the `@override` directive no longer exists in a source subgraph, the directive can be safely removed."
>

The following example violates the rule:
<br/>

```graphql title="❌ Subgraph A" disableCopy=true showLineNumbers=false {3}
type Product @key(fields: "id") {
  id: ID!
  inStock: Boolean! @override(from: "Subgraph B")
}
```

```graphql title="❌ Subgraph B" disableCopy=true showLineNumbers=false
type Product @key(fields: "id") {
  id: ID!
  name: String!
}
```

<br/>
Use instead:
<br/>

```graphql title="✅ Subgraph A" disableCopy=true showLineNumbers=false {3}
type Product @key(fields: "id") {
  id: ID!
  inStock: Boolean!
}
```

```graphql title="✅ Subgraph B" disableCopy=true showLineNumbers=false
type Product @key(fields: "id") {
  id: ID!
  name: String!
}
```

</RuleExpansionPanel>

</RuleExpansionPanel>

<RuleExpansionPanel
  title="UNUSED_ENUM_TYPE"
  whatItDoes="Checks if an enum type is defined but no field or argument in any subgraph references it."
  rationale="If the enum is defined, it should likely be used or removed."
>

The following example violates the rule:
<br/>

<CodeColumns>

```graphql title="❌ Subgraph A" disableCopy=true showLineNumbers=false
enum ProductStatus {
  AVAILABLE
  SOLD_OUT
}

type Product {
  id: ID!
  name: String
}
```

```graphql title="❌ Subgraph B" disableCopy=true showLineNumbers=false  {4}
type Order {
  id: ID!
  product: Product
  status: String
}
```

</CodeColumns>

<br/>
Use instead:
<br/>

<CodeColumns>

```graphql title="✅ Subgraph A" disableCopy=true showLineNumbers=false

type Product {
  id: ID!
  name: String
  status: ProductStatus
}
```

```graphql title="✅ Subgraph B" disableCopy=true showLineNumbers=false {4}

type Order {
  id: ID!
  product: Product
  status: ProductStatus
}
```
</CodeColumns>

</RuleExpansionPanel>
